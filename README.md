# Student Attendance Monitoring System

A JavaFX-based desktop application for managing students, courses, attendance records, and attendance reports in an academic environment.

## Project Overview

The Student Attendance Monitoring System is designed to help teachers and academic administrators. The system provides a simple graphical interface for managing student information, course details, daily attendance marking, and generating attendance reports.

This project was developed as part of an Object-Oriented Programming practical project using JavaFX, MySQL, JDBC, and Maven.

## Main Features

- Student record management
- Course management
- Attendance marking
- Attendance report generation
- Input validation
- MySQL database connectivity
- JavaFX graphical user interface

## Technologies Used

- Maven
- MySQL
